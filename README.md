cwb assistant
=================

An exercise is creating a mobile hybrid app using the Ionic Framework. 

##### Get started

##### Install NodeJS (duh)

Nodejs [http://nodejs.org]

##### Install Ionic

```
npm install -g ionic
```

##### Install Dependencies

```
npm install
```

```
bower install
```

##### Run locally

```
ionic serve
```

##### Run tests

```
npm test
```
